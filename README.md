# UPS-1S-LiPo / easyeda-fragment

Этот репозиторий содержит тестовый фрагмент схемы EasyEDA для компактного, высокоэффективного блока питания (1S Li‑ion → 3.3V) для модуля ME25LS02.

Цель
- Подготовить готовый к импорту в EasyEDA JSON‑фрагмент схемы (BUCK + обвязка) для быстрого прототипа.
- Дать BOM и рекомендации по размещению (layout) и тестированию.

Содержимое
- ME25LS02_power_fragment_easyeda.json — EasyEDA JSON‑фрагмент схемы (готов к импорту/вставке в новый проект).
- README.md — этот файл.
- .gitignore — базовые игнор правила.

Как использовать (рекомендации)
1. Сохраните `ME25LS02_power_fragment_easyeda.json` на диск.
2. В EasyEDA: создайте новый проект → Schematic → File → Import → Schematic File (если доступно) и выберите JSON.  
   - Если импорт не срабатывает, используйте Place → Text и вставьте JSON как заметку, либо откройте пустой проект и используйте Edit Raw Data (если доступно) и вставьте содержимое JSON.
3. После импорта замените placeholder‑компоненты (если они есть) на реальные элементы из библиотеки: TPS62743 (или эквивалент) для BUCK_IC, 0805 inductor, 0603/0805 capacitors, 0603 ferrite bead.
4. Проверьте net‑имена: VOUT_PROT, BUCK_IN, BUCK_SW, BUCK_OUT, VCC_MODULE, GND.
5. Convert → PCB и расположите компоненты согласно рекомендациям (минимизировать петли VIN→CIN→GND и SW→L→Cout, ferrite рядом с модулем, keepout для антенны).

BOM (кратко)
- U1: TPS62743 (фикс. 3.3V) WSON-10 (рекомендуется)
- L1: 0.47µH 0805 (Isat >= 1A)
- C_IN: 4.7µF 0805 X5R
- C_IN2: 100nF 0603 X7R
- C_OUT: 10µF 0603 X5R
- C_OUT2: 100nF 0603 X7R
- FB1: Ferrite bead 0603 (~120Ω@100MHz)
- Доп.: TVS, PTC (опционально)
