# UPS 1S LiPo — EasyEDA project (ru)

Описание
- Компактный UPS/зарядный модуль для 1S Li‑Po (заряд до 4.20 V), выход 3.3 V стабилизированный, нагрузка до 150 mA.
- Ключевые параметры:
  - Аккумулятор: 1S Li‑Po, 300 mAh.
  - Зарядный ток: 90 mA (Rprog ≈ 13.3 kΩ).
  - Вход: паяльные площадки VIN (pads) — без встроенного разъёма.
  - На плате предусмотрено место/footprint для внешнего PCM‑модуля (DW01 + FS8205A).
  - Зарядный контроллер: MCP73871 (power‑path).
  - Buck‑boost регулятор: TPS63070 (выход 3.3 V).
  - Индикаторы: LED CHG (STAT), LED SYS (POWER OK).
  - Polyfuse на BAT+ для защиты по току.
- Форма платы: 2 слоя, целевой размер 40 × 30 mm, 2 монтажных отверстия Ø2.5 mm.

Содержимое проекта
- Схема / netlist (human readable)
- PCB layout (2 слоя) — черновая разводка (я могу сделать разводку в EasyEDA/загрузить файлы)
- BOM (CSV) с рекомендациями по компонентам и ссылками на EU‑дистрибьюторов
- Pick&Place (CSV)
- Board notes (размещение и рекомендации по разводке)
- Инструкция как сгенерировать Gerber и заказать плату

Рекомендации по использованию
- Rprog рассчитан для Icharge ≈ 90 mA: Rprog_kΩ ≈ 1200 / I(mA) → 1200/90 ≈ 13.33 kΩ → используйте 13k3 (E96 1%).
- Используйте защищённую батарею или припаивайте внешний PCM‑модуль на предусмотренное место.
- Тестирование: подайте 5 V на VIN pads, проверьте STAT LED и SYS LED, убедитесь в 3.3 V на выходе OUT_pad при подключённой батарее (и без батареи).
- Лицензия: MIT (по желанию).
